___
### **입력**  
 > **첫째 줄에 \(N\)과 \(K\)가 주어진다. (1 ≤ \(N\) ≤ 10, 0 ≤ \(K\) ≤ \(N\))**
 
 ___
### **출력**  
 > **\(\binom{N}{K}\)를 출력한다.**
